\hypertarget{index_intro}{}\doxysection{Program Introduction}\label{index_intro}
The {\bfseries{TMS}} project is meant to revolutionize the shippign and logistics industry. Our project allows for the purchasing, scheduling, monitoring and billing of organizations who want to have cargo shipped via truck or train. Though this work is currently in a prototype stage, it is meant to model the effectiveness of using a TMS solution as opposed to more traditional shipping providers.
